Project Overview

This project is a simple and responsive Task Management Web Application designed to help users organize their daily tasks efficiently. The application allows users to add, manage, and track tasks through an intuitive and user-friendly interface.

Key Features

  • Add new tasks dynamically using a form
  • Display tasks in an organized list
  • Edit and delete existing tasks
  • Mark tasks as completed
  • Sort tasks by priority or due date
  • Responsive design for desktop and mobile screens
  • Clean and simple user interface
  • Light / Dark mode
  • Search or filter tasks
  • LocalStorage support to save tasks

My Role

I was responsible for the full development process, including:

  • Designing the HTML structure
  • Styling the UI with CSS
  • Implementing all interactive features using JavaScript
  • Ensuring responsiveness and usability

Challenges & Solutions

One of the main challenges was handling dynamic task updates (add, edit, delete) efficiently using JavaScript. This was solved by organizing the code into clear functions and using DOM manipulation properly.
