Project Overview

The Super Market Management System is a Windows Forms (WinForms) desktop application developed to manage supermarket operations efficiently using a connected database. The system provides a user-friendly interface that allows administrators and staff to manage product data and other supermarket entities in an organized and centralized way. The application enables users to view, search, add, update, and delete product records directly from the database. It is designed to improve accuracy, reduce manual work, and ensure smooth management of supermarket data such as products, customers, stock, payments, suppliers, departments, and employees.

Key Features

  • Product Management
    • Display all products in a structured data grid.
    • Add new products with details such as Product ID, Name, Class ID, Price, Brand, Status, Expiry Date, Department ID, and Quantity.
    • Update existing product information.
    • Delete products from the database.
  • Search Functionality
    • Search for products quickly using the search bar.
  • Database Integration
    • All data is stored and retrieved from a database, ensuring data consistency and reliability.
  • User-Friendly Interface
    • Simple and clear WinForms UI for easy navigation.
    • Side menu for quick access to different modules (Product, Customer, Stock, Payment, Department, Suppliers, Employee).
  • Data Validation
    • Ensures correct input when adding or updating product information.
  • Efficient Data Handling
    • Load and refresh data easily using the Load button.
